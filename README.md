#### 核心玩法：
> 上下左右键控制走向，吃到10颗果实赢得胜利

> 撞击墙壁、自身都会失败

>【ctrl + c】退出游戏

#### 贪吃蛇原理：
> 贪吃蛇在移动的时候是将尾部元素放到头部来实现移动。

> 吃到果子不需要挪到尾部，而是在头部新增一个节点。
> 这和 list 结构就很像了，所以 snake 的本体结构我用 list 来描述。
> 渲染部分用的是数组，方便随机寻址。

#### 实现难点：
* 键盘事件的捕捉
* 渲染粒度的把控，关系到实现难度
* 贪吃蛇核心玩法，对象结构设计
